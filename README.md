# Опис
Програма, що реалізує онлайн чат за допомогою технології вебсокетів.

# Документація

### Програмна документація
Програма написана на TypeScript, з використанням фрейморка Express js, та бібліотек Mongoose, Socket.io, Express-session.

Комунікація з клієнтами здійснюється за допомогою протоколу вебсокетів.



Весь функціонал програми знаходиться у папці ***src***.

***src/app.ts*** - містить всі налаштування сервера, та його логіку обробки запитів.

***src/core/Objects*** та ***src/core/Services*** - містить сервіси та об'єкти даних, які потрібні для функціонування сервера.



# Встановлення
Для встановлення, відкрийте `Git bash` у папці, в яку потрібно завантажити цей проєкт, та виконайте наступну команду

```
git clone https://github.com/ArtVitLobanov/WebSocketChat
```

### Запуск
***Переконайтеся, що у вас встановлені `.Node.js 25.0`, `Express js 5.2.0`, `Socket.io 4.8.0`, `Express-session 1.19.0` та вище!***

Для запуску програми, перейдіть у завантажений проєкт, скомпілюйте TypeScript файли у JavaScript та в директорії проєкту виконайте команду:
```
npm run full
```
