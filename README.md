# Опис
Програма, що реалізує онлайн чат за допомогою технології вебсокетів. Присутня як backend так і frontend частина.

# Документація

### Програмна документація
Серверна частина програми написана на TypeScript, з використанням фреймворка Express js, та бібліотек Mongoose, Socket.io, Express-session.
Клієнтська частина програми написана на TypeScript, з використанням фреймворка Angular, та бібліотеки Sokect.io (а саме її частиною - socket.io-client). 

Комунікація між серверною та клієнтською частино здійснюється за допомогою протоколу вебсокетів.



Весь функціонал серверної частини програми знаходиться у директорії ***server/src/***.
 - ***app.ts*** - містить всі налаштування сервера, та його логіку обробки запитів.
 - ***core/Objects*** та ***src/core/Services*** - містить сервіси та об'єкти даних, які потрібні для функціонування сервера.



Весь функціонал клієнтської частини програми знаходиться у директорії ***client/front-angular/***
 - ***src/app*** - містить всі налаштування клієнтської частини додатка: логіку компонентів; їх розміщення на сторінці; їх стильове оформлення.
 - ***src/app/services*** - містить сервіси, необхідні для зв'язку компонентів з сервером.


# Встановлення
Для встановлення, відкрийте `Git bash` у папці, в яку потрібно завантажити цей проєкт, та виконайте наступну команду

```
git clone https://github.com/ArtVitLobanov/WebSocketChat
```

### Запуск
***Переконайтеся, що у вас встановлені `.Node.js 25.0`, `Express js 5.2.0`, `Socket.io 4.8.0`, `Express-session 1.19.0`, `Angular 21.2.0`, `socket.io-client 4.8.3` та вище!***

Для серверної частини програми, перейдіть у завантажений проєкт. Далі перейдіть у директорію ***server/***  скомпілюйте TypeScript файли у JavaScript та в директорії проєкту виконайте команду:
```
npm run full
```

Для клієнтської частини програми, перейдіть у завантажений проєкт. Далі перейдіть у директорію ***client/front-angular/***  скомпілюйте TypeScript файли у JavaScript та в директорії проєкту виконайте команду:
```
ng serve
```
